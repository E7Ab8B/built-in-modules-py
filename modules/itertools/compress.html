compress()#

https://docs.python.org/3/library/itertools.html#itertools.compress

Syntax#

from itertools import compress

compress(data, selectors)

Description#

Filters elements from an iterable based on the corresponding values in a selector iterable.
Elements are selected for inclusion in the output if the corresponding value in the selector evaluates to True.

Usage#

  • Filtering Sequences

Examples#

Example 1#

from itertools import compress

values = "ABCDE"

compressed_values_seq = compress(values, [1, 0, 1, 0, 1])

"".join(compressed_values_seq)

'ACE'

Example 2#

from itertools import compress, cycle

values = [0, "", None, 1, 2, "hello"]

compressed_values_seq = compress(values, cycle([1, 0]))

list(compressed_values_seq)

[0, None, 2]
